# nearup
Public scripts to launch near devnet, betanet and testnet node

# Usage
## Install
```
curl -sL up.nearprotocol.com | python3
```

## Start devnet, betanet, testnet
```
nearup devnet
nearup betanet
# WIP
# nearup testnet
```

## Stop a running node
```
nearup stop
```

## Additional options
```
nearup {devnet, betanet, testnet} --help
```
